Iris-isstible

Controlez ce que vos yeux révèlent

Les émotions sont parfois difficiles à gérer ou à montrer. Pourtant nos yeux nous trahissent souvent.

Grâce à Iris-isstible et aux lentilles connectées, vous choisissez ce que vos yeux expriment. L'application vous donne le pouvoir de rester maître de votre image émotionnelle.

Plus de contrôle. Moins de malentendus. une nouvelle manière de communiquer.

Les fonctionnalités

Notre application mobile vous permet de modifier en un geste la couleur de votre lentille connectée afin qu’elle reflète votre humeur du moment. Joie, sérénité ou énergie : personnalisez votre regard à tout instant, directement depuis votre smartphone.

Objectif de l'app

Adapter la couleur de la lentille à l'humeur que vous souhaitez partager

L'application

SETUP

Voici la première page de cette application. Sur cette page, vous trouverez un premier bouton qui vous mènera à la page suivante, le scan. Si vous n'avez pas encore de compte vous pourrez également accéder à la création du compte via le lien.

Scannez votre iris pour laisser parlez vos émotions.

Ici, vous pourrez choisir le type d'émotion que vous voulez, plutôt positive, plutôt négative ou encore une émotion dite mixte. Vous pourrez également choisir l'émotion via le champ de recherche ou via vos émotions likées.

FEEL

Ensuite, sur cette page, vous trouverez le nom de l'émotion, sa couleur, un descriptif, ainsi qu'une visualisation de votre lentille. Vous trouverez aussi le nombre de fois où cette émotion est apparue dans le mois. Il y a aussi un boutton pour ajouter l'émotion aux favoris et l'autre pour activer.

Retrouvez vos émotions jour après jour.

USR

Accédez à vos informations rapidement.

Ici, vous pourrez choisir le type d'émotion que vous voulez, plutôt positive, plutôt négative ou encore une émotion dite mixte. Vous pourrez également choisir l'émotion via le champ de recherche ou via vos émotions likées.
